Theory (Key Concepts)

Matplotlib – Basic plotting library

plt.plot() → Line plots

plt.bar() → Bar charts

plt.hist() → Histograms

plt.scatter() → Scatter plots

Customizations: titles, labels, legends, colors

Seaborn – Built on Matplotlib, higher-level visualizations

sns.histplot(), sns.countplot() → Frequency plots

sns.boxplot() → Statistical plots

sns.heatmap() → Correlation heatmaps

sns.pairplot() → Visual relationships between multiple features
