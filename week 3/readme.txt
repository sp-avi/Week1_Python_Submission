Theory (Key Concepts)

NumPy

Arrays: Efficient way to store numerical data (np.array)

Operations: Element-wise addition, multiplication, etc.

Broadcasting: Apply operations on arrays of different shapes

Useful functions: np.mean(), np.sum(), np.reshape(), etc.

Pandas

Series: 1D labeled array

DataFrame: 2D labeled data structure (like Excel table)

Indexing & selection: .loc[], .iloc[], filtering rows/columns

Grouping: .groupby() to aggregate data
